# 校园自习室预约系统 - 前端客户端 (Frontend)

基于 **Vue 3** + **Vite** + **TypeScript** 构建的校园自习室预约系统前端客户端，集成了 Element Plus 组件库、Pinia 状态管理、Axios 请求库以及 Tailwind CSS 4.0 高级设计系统。

## 🛠️ 环境要求

- **Node.js**: v18.0.0+ 或 v20.0.0+（推荐使用较新的 LTS 版本）
- **包管理器**: npm / yarn / pnpm（推荐使用 `pnpm`）

---

## 🚀 快速启动指南

### 第一步：安装依赖项
在当前 `frontend` 目录下打开终端，执行以下指令安装项目依赖：
```bash
npm install
# 或者如果您使用 pnpm (推荐)
pnpm install
```

### 第二步：运行本地开发服务器
执行以下命令启动 Vite 开发服务器：
```bash
npm run dev
# 或者使用 pnpm
pnpm dev
```
启动成功后，控制台会输出本地访问地址，默认在浏览器中访问：
🔗 **本地服务地址**: [http://localhost:5173](http://localhost:5173)

### 第三步：构建生产环境包
如果您需要将项目进行部署，可运行以下打包命令，将在 `dist` 目录下生成静态资源文件：
```bash
npm run build
# 或者使用 pnpm
pnpm build
```

---

## 🔗 后端联调与反向代理

在开发模式下，项目的代理配置已在 `vite.config.ts` 中设定：
```typescript
server: {
  port: 5173,
  proxy: {
    '/api': {
      target: 'http://localhost:8080', // 指向本地后端服务默认端口
      changeOrigin: true
    }
  }
}
```
* **无缝启动**：只要确保后端程序运行在本地的 `8080` 端口上，前端的所有 `/api` 请求都会被自动转发，无需做额外的跨域配置（CORS）或硬编码 IP。

## 🌟 核心特性与技术亮点

1. **AI 智能推荐轮询**：学生端自习室推荐采用 500ms 异步轮询任务结果（最多轮询 10s 并具备骨架屏加载态）。若后端响应超时或降级，前端会自动优雅转向同步请求防白屏。
2. **预约时间安全联动**：
   - 预约开始时间自动根据当前分针（30分分水岭）向上取整，且结束时间与开始时间进行 `开始 + 1小时` 的双向动态响应式绑定。
   - 过滤了已被占用的时间段与过去时间置灰，保障数据输入的合法性。
3. **全局异常单次提示**：移除多余的重复 Toast 警示框，统一通过 Axios 响应拦截器进行集中处理，带给用户更清爽的界面交互。
4. **管理端删除二次确认**：对自习室和座位的“直接删除”操作加入 `ElMessageBox.confirm` 二次弹窗拦截，防止意外删除引起大范围预约退单。
