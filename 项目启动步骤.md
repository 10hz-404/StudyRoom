### 🛠️ 第一步：新电脑的基础软件准备

确保新电脑上已经安装了以下四款开发必备软件：

1. **Docker Desktop**（用于一键跑起数据库和中间件，免去手动安装 MySQL/Redis/RabbitMQ 的麻烦）
2. **JDK 21**（后端运行环境）
3. **Node.js (推荐 v20.x 或以上)**（前端运行环境）

------

### 🗄️ 第二步：一键启动基础设施 (数据库/缓存/队列)

打开终端，进入项目中的 **`docker`** 文件夹，执行命令启动所有服务：

```bash
# 1. 进入 docker 目录
cd docker
# 2. 启动底层容器（会自动创建并导入 study_room 数据库及测试数据）
docker-compose up -d
```

- **如何验证启动成功**：在终端输入 `docker ps`，看到 `studyroom-mysql`、`studyroom-redis`、`studyroom-rabbitmq` 都在运行（Up状态）即可。

------

### ☕ 第三步：启动后端服务 (Backend)

在编辑器中打开 `code/backend` 目录，或者在终端中执行：

```bash
# 1. 进入后端代码目录
cd ../code/backend
# 2. 在application.yml文件中配置好DeepSeek APIKEY
api-key: ${DEEPSEEK_API_KEY:你的APIKEY放这里}
# 3. 用 Maven 编译并打包（自动下载依赖包）
mvn clean package
# 4. 运行后端服务
java -jar target/study-room-backend-1.0.0.jar
```

- **如何验证启动成功**：控制台输出 `Started StudyRoomApplication in ... seconds`，并且可以在浏览器访问 http://localhost:8080/doc.html 查看 Knife4j API 接口文档。

------

### 💻 第四步：启动前端项目 (Frontend)

重新打开一个终端窗口，进入 `code/frontend` 目录，执行以下步骤：

```bash
# 1. 进入前端代码目录
cd code/frontend
# 2. 安装前端依赖包（推荐使用 pnpm，若无 pnpm 也可直接用 npm）
npm install
# 3. 启动 Vite 开发服务器
npm run dev
```

- **如何验证启动成功**：控制台会输出 `Local: http://localhost:5173/`。

------
